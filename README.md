# Alumni Learning Tracker

A blockchain-based lifelong learning platform that rewards alumni for continuing their education.

## Overview

The Alumni Learning Tracker incentivizes continuous learning by rewarding alumni with points for completing courses. Built on the Stacks blockchain, it provides transparent tracking of educational achievements.

## Features

- Alumni registration system
- Course creation and management
- Completion tracking with rewards
- Point-based incentive system
- Transparent achievement history

## Smart Contract Functions

### Public Functions
- `register-alumni` - Register as an alumni user
- `create-course` - Create a new course (owner only)
- `complete-course` - Mark a course as completed and earn rewards

### Read-only Functions
- `get-course` - Retrieve course information
- `get-alumni-profile` - View alumni profile and points
- `get-total-courses` - Get total number of courses
- `has-completed-course` - Check if alumni completed a course

## Getting Started

1. Deploy the contract to Stacks blockchain
2. Register as an alumni user
3. Complete courses to earn reward points
4. Track your lifelong learning journey
